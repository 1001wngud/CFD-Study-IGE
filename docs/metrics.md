# Metrics (how to compute)

## 1) Thrust ratio
T/Tref
- T: propellerDisk가 출력하는 force 벡터를 rotor normal 방향으로 투영한 값
- Tref: h/R = 4 케이스의 T

## 2) Power ratio
P/Pref, P = 2π n Q
- n: propellerDisk 입력 회전수 [1/s]
- Q: propeller axis(normal) 방향 moment 성분
- Pref: h/R=4 케이스의 P

## 3) Outwash peak location
rmax/R
- 샘플링 높이: z ≈ 0.02 R
- radial profile에서 |Ur(r)| 또는 |U|(r)의 최대점 rmax를 찾고 rmax/R로 기록
